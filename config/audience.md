## 深層心理

### 恐れ（これがあるから行動しない）

| 恐れ | 心の声 | 台本での対処 |
|------|--------|-------------|
| 難しそう | 「若い人向けでしょ」 | 「ズボラでもできる」と保証 |
| 自分には無理 | 「覚えられない」 | 「設定するだけ」「1回やればOK」 |
| 失敗したら恥ずかしい | 「周りに聞けない」 | 手順を丁寧に見せる |
| 時間がかかりそう | 「面倒くさい」 | 「3分で」「5ステップで」 |

### 欲求（これがあるから見る）

| 欲求 | 心の声 | 台本での対処 |
|------|--------|-------------|
| 簡単にできると保証してほしい | 「本当に私でも？」 | 「私みたいなズボラでも」 |
| 成功体験を得たい | 「できた！と言いたい」 | 再現可能な手順を提示 |
| 知識を得たい | 「知ってる側になりたい」 | 「実は〇〇なんです」 |
| 無駄を減らしたい | 「もっと楽したい」 | 効果を具体的に示す |

### 信頼のトリガー（これがあるから信じる）

```
1. 同じ「面倒くさがり」だと共感できる
   → 「私もズボラで〜」という自己開示

2. 具体的な手順を見せてくれる
   → 抽象論ではなく「〜をタップ」レベル

3. 「検証済み」という安心感
   → 「私が検証して本当に良かったもの」

4. デメリットも正直に言う
   → 「ただし〜には注意」で信頼構築
```

## 視聴行動パターン

### YouTube内での行動

```
・サムネイルの「簡単」「ズボラ」に反応する
・タイトルに数字があると具体性を感じる
・最初の30秒で「自分向けか」を判断する
・飛ばし見より順番に見る傾向
・構造化された情報（STEP形式）を好む
```

### 動画視聴後の行動

```
・その場で試してみたい
・メモを取りながら見ることもある
・家族に教えたい（知識共有欲）
・コメントで質問することは少ない
```

## 言語感覚

### 響く言葉

```
✅ 「簡単」「楽」「ズボラ」「すぐできる」
✅ 「設定するだけ」「これだけで」
✅ 「〇〇選」「〇分で」「〇ステップ」（数字）
✅ 「知らないと損」「実は〜」
✅ 「私みたいな人でも」
```

### 響かない言葉

```
❌ 「やばい」「神」「最強」「マジで」（若者言葉）
❌ 「爆速」「秒で」（誇張表現）
❌ 「情弱」「知らない人は〜」（見下す表現）
❌ 専門用語そのまま（API、UI、UXなど）
```

### 専門用語の言い換え

| 専門用語 | 言い換え |
|----------|----------|
| アーカイブ | 「消さずにしまっておく箱」のようなもの |
| UI | 画面の見た目・操作のしやすさ |
| 同期 | 自動で同じ状態に揃える |
| クラウド | インターネット上の保存場所 |
| 2段階認証 | パスワードに加えてスマホでも本人確認する仕組み |

## 導入30秒の心理設計

```
0-5秒：   「誰が話してるの？」→ 自己紹介で解決
5-10秒：  「自分に関係ある？」→ 問いかけで自分事化
10-20秒： 「何が得られる？」→ 価値リストで約束
20-30秒： 「見続ける価値ある？」→ 宣言で確信
```

この30秒で離脱を防げないと、どんなに良い内容でも届かない。

## セクション設計の心理

### なぜSTEP形式が効くのか

```
視聴者心理：
・「今どこにいるか」がわからないと不安
・「あとどれくらいか」がわからないと離脱
・進捗感がないと「長い」と感じる

STEP/番号形式の効果：
・「STEP1, 2, 3」→ 現在地が明確
・「6選」→ ゴールが見えている
・1つ終わるごとに達成感
```

### 40-60代の情報処理特性

```
・若い世代より「構造化された情報」を好む
・「飛ばし見」より「順番に見る」傾向
・明確な区切りがあると理解しやすい
・一度に多くの情報を詰め込まない方が良い
```
