# 文体・トーン（SSOT）

> 執筆エージェントはこのファイルに基づいて台本を書きます。
> 推敲比較（/推敲比較）で差分を直すときは、言葉の問題に限らず brand・構成・品質など該当する SSOT へ反映する。手順は [agents/calibrator.md](../agents/calibrator.md) と [config/quality.md](quality.md) の推敲比較の節を参照。

---

## 基本ルール

| 項目 | ルール |
|------|--------|
| 一人称 | 「私」または「わたし」 |
| 敬語 | 丁寧語（です・ます）、堅すぎない |
| 口語表現 | 適度に砕ける（「なんで」「ちょっと」「けっこう」） |
| 操作説明 | PC:「クリック」、スマホ:「タップ」 |

---

## トーンの3要素

### 1. 親しみやすさ

```
・「私みたいなズボラな人って」と自分を下げて共感を誘う
・「〜ですよね」「〜って感じです」と柔らかい語尾
・堅い説明の後に「これでOKです」と砕ける
```

### 2. 信頼感

```
・「実は〜なんです」と専門知識を見せる
・「ただし〜には注意してください」とデメリットも伝える
・「私が検証して〜」と実体験を示す
```

### 3. 行動促進

```
・「是非試してみてください」と明確に行動を促す
・「これだけで〜ができます」と効果を言語化
・「〇〇な人は」と対象を明確にして自分事化
```

## 頻出フレーズ集

### 導入系

```
「『楽』を極めて最大の結果を出す、効率化オタクのあらきりです」（固定）
「是非最後まで見てみてください」
「〜な人に向いています」
```

### 展開系

```
「では早速〜」
「続いて〜も見ていきましょう」
「ここがポイントなんですけど」
「試しに〜してみます」
「こんな風に〜することで」
「実際に〜してみると」
```

### 共感系

```
「〜な気持ちはよくわかります」
「やっぱり、私みたいなズボラな人って」
「特にズボラな人にとっては」
「面倒くさいと感じる原因は、手間の多さです」
「私もそうだったんですけど」
```

### 補足系

```
「ただし〜なので注意してください」
「個人的には〜が良いと思います」
「これで〜ができるようになります」
「ちなみに〜」
「もし〜な場合は」
```

### 理由説明系

```
「これ、なぜかというと〜」
「実は〜なんです」
「どうして〜かというと」
「〜には理由があって」
```

### 先を見せる系

```
「その理由は後半でお伝えしますね」
「詳しくは別の動画で紹介しています」
「ここが一番の正念場です」
```

### まとめ系

```
「というわけで今回は〜」
「ポイントは〜つあります」
「是非試してみてください」
「たったこれだけで〜」
「それでは、また次回の動画でお会いしましょう。さようなら。」（固定）
```

## 操作説明の書き方

### 基本形

```
「〜を【動詞】します」
「〜を【動詞】してください」
```

### 動詞の使い分け

| デバイス | 動詞 |
|----------|------|
| PC | クリック、押す、入力する |
| スマホ | タップ、押す、入力する、スワイプする |
| 共通 | 選ぶ、選択する、確認する、開く |

### 良い例

```
✅ 「設定から位置情報をタップしてください」
✅ 「右上のアイコンを押して『Googleアカウントを管理』を選びます」
✅ 「すると、こんな風に〜が表示されます」
```

### 悪い例

```
❌ 「設定画面にアクセスしてください」（抽象的）
❌ 「位置情報を有効化します」（専門的）
❌ 「そこから進んでください」（曖昧）
```

## 3層構造の書き方

### Layer 1: 結論（1-2文）

```
【パターン】
「〜は、〜ができる機能/サービスです」
「〜を使うと、〜ができます」

【例】
「Amazon Photosは、追加料金なしで写真を無制限に保存できる特典です」
「スワイプを設定すると、メールの仕分けが一瞬で終わります」
```

### Layer 2: 具体的手順（3-10文）

```
【パターン】
「まず〜をタップします」
「続いて〜を選択します」
「すると〜が表示されます」
「試しに〜してみます」

【例】
「まず、サイドメニューから設定を開きます」
「続いて、受信トレイのカスタマイズをタップします」
「すると、こんな風にカテゴリの設定画面が表示されます」
```

### Layer 3: 補足・注意点（1-3文）

```
【パターン】
「ただし、〜なので注意してください」
「個人的には〜がおすすめです」
「これで〜ができるようになります」

【例】
「ただし、無制限なのは写真だけで、動画は5GBまでです」
「個人的には『どこでもネットワークを使用する』が良いと思います」
「これで、新着メールがひと目でわかるようになります」
```

## 避けるべき表現

| 避ける表現 | 理由 | 代替表現 |
|------------|------|----------|
| 「やばい」「神」「最強」 | 40-60代に響かない | 「かなり便利」「おすすめ」 |
| 「爆速」「秒で」 | 誇張しすぎ | 「すぐに」「一瞬で」 |
| 「情弱」「知らない人は損」 | 見下している | 「知っておくと便利」 |
| 「〜しないと大変なことに」 | 不安を煽る | 「〜しておくと安心」 |
| 「当然ですが」「常識ですが」 | 上から目線 | 「ご存知の方も多いと思いますが」 |

## 文の長さ

```
・1文は40-60字程度を目安
・長くなったら「。」で区切る
・接続詞で繋げすぎない
```

### 良い例

```
✅ 「アーカイブはメモやデータの『保管庫』のようなものです。
    もう頻繁には使わないけれど、削除はしたくない情報を残しておくための保存方法です。」
```

### 悪い例

```
❌ 「アーカイブはメモやデータの『保管庫』のようなもので、もう頻繁には使わないけれど削除はしたくない情報を残しておくための保存方法で、ゴミ箱と違って自動的に削除されることはありません。」
```
