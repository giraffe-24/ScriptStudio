## ブランドコア（3つの柱）

### 1. 「楽して結果を出す」の哲学

```
・努力より仕組み化を重視
・設定すれば自動で動く
・最小の手間で最大の効果
```

**台本での表現例：**
- 「設定すれば自動で〜」
- 「これだけで〜」
- 「たったこれだけの管理で」

### 2. 「ズボラでもできる」の簡単さ

```
・ハードルを徹底的に下げる
・「難しそう」を「自分にもできそう」に変える
・面倒な設定を省く or 簡略化する
```

**台本での表現例：**
- 「私みたいなズボラな人でも」
- 「面倒な設定なし」
- 「ズボラを極めたような人間」

### 3. 「40-60代にもわかる」丁寧さ

```
・専門用語は必ず説明する
・手順は「〜をタップ」レベルまで具体化
・抽象的な説明を避ける
```

**台本での表現例：**
- 「アーカイブはメモやデータの『保管庫』のようなものです」
- 「〜をタップしてください」「〜をクリックします」

## ペルソナ：あらきり

```
・効率化オタク（自称）
・ズボラを公言する（視聴者との距離を縮める戦略）
・自分で検証したものだけを紹介する誠実さ
・「楽」を追求する哲学を持つ
・実際の失敗体験も共有する（信頼構築）
```

## 「ズボラ」の戦略的意味

```
一般的なYouTuber：
「すごい人」が「すごいこと」を教える
→ 視聴者は「自分には無理」と感じる

あらきり：
「ズボラな人」が「ズボラでもできること」を教える
→ 視聴者は「自分にもできそう」と感じる
```

**本質：**
- 「ズボラ」は自己卑下ではなく、視聴者との距離を縮める戦略的ポジショニング
- 「私でもできた」→「あなたにもできる」の論理構造
- ハードルを下げることで行動を促す
