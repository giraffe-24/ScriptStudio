# 台本テンプレート A：ツール活用型

> 1つのツールを深掘りし、ズボラ向けの活用法を紹介する型
> 例：Gmail管理術、Googleカレンダー×Keep×ToDo

---

## 適用条件

```
・1つのツール/サービスを深掘りする
・「ズボラでもできる」設定や使い方を紹介
・手順が複数ステップある
・視聴者の「面倒くさい」を解決する
```

---

## 構成

```
【導入】約300字（10-15%）
  │
  ├── 自己紹介（固定）
  ├── 共感フック「私はとにかく〜が面倒くさくて」
  ├── 課題提示「〜が溜まって」「〜で困っていて」
  ├── 解決提示「今回紹介する方法なら〜」
  └── 視聴継続「ズボラな私でも実施できているので、是非最後まで」

【本題】約1500-2000字（70-75%）
  │
  ├── STEP1：〇〇する
  │     ├── 結論（何をするか）
  │     ├── 手順（具体的な操作）
  │     └── 効果（これで何ができるか）
  │
  ├── STEP2：〇〇する
  │     ├── 結論
  │     ├── 手順
  │     └── 効果
  │
  ├── STEP3：〇〇する
  │     ├── 結論
  │     ├── 手順
  │     └── 効果
  │
  ├── STEP4：〇〇する（必要に応じて）
  │
  └── STEP5：追加の管理は継続できる範囲で（任意）
        └── 「慣れてきたら〜もアリ」で応用を示す

【まとめ】約300字（10-15%）
  │
  ├── 振り返り「というわけで今回は〜を紹介しました」
  ├── ポイント整理（3つ）
  ├── 価値再確認「たったこれだけで〜」
  ├── 行動喚起「〜な人は、是非試してみてください」
  └── 定型締め（固定）
```

---

## 導入テンプレート

```
「楽」を極めて最大の結果を出す、効率化オタクのあらきりです。

今回は【ターゲット】必見！【ツール名】を直感的に、超簡単に【できること】できる最強管理方法を紹介します。

私はとにかく【課題】が面倒くさくて、【結果】してしまいます。
【具体的な困りごと】こともしばしば。

ですが、今回紹介する管理方法なら【得られる結果】ができてスッキリします。

ズボラな私でも実施できているので、是非最後まで見てみてください。
```

---

## 本題テンプレート（1セクション）

```
【見出し】STEP【数字】：【動詞で始まる行動】

【結論】1-2文
今回紹介する管理方法では、【何をするか】をしていきます。
【ツール名】は多機能なので、今回は【使う機能】のみを使っていきます。

【手順】3-8文
まず【操作1】をします。
【場所】を見ると、【表示されるもの】がありますね。
【操作2】をタップ/クリックすると、【結果】が表示されます。
試しに【操作】してみます。

【補足・効果】1-3文
これで【得られる効果】ができます。
やっぱり、私みたいなズボラな人って、【共感ポイント】って思うんですけど、
【この設定のメリット】することで、【改善される点】になります。
```

---

## まとめテンプレート

```
【振り返り】
今回は【テーマ】を紹介しました。

【ポイント整理】
【ツール名】って無料なのに多機能なんで、いろんな設定に目が行きがちです。

結局【本質的なメリット】ことが一番大切です。

【ツール名】の機能をうまく活用すれば、【具体的な効果1】がなくなり、【具体的な効果2】が一瞬で終わります。

【結果の状態】になるので、【メリット】ができます。

たったこれだけの管理で、嬉しいメリットが沢山あります。

【行動喚起】
【対象者】は、是非試してみてください。

【定型締め】
このチャンネルでは「楽」を極めるために私が検証し、本当に効果があった仕組みだけを紹介しています。

大人の教養として『賢くサボる』を知りたい方は、ぜひチャンネル登録と高評価をお願いします。

日常に潜む「無駄な手間」や｢これも知りたい｣などのリクエストも、遠慮なくコメントで教えてください。

それでは、また次回の動画でお会いしましょう。さようなら。
```

---

## セクション見出しの例

```
・STEP1：使わないものはできるだけ非表示に
・STEP2：スワイプの設定をする
・STEP3：受信ボックスを空にする
・STEP4：新着メールを振り分ける
・STEP5：追加の管理は継続できる範囲で
```

## この型の成功パターン

### Gmail台本から抽出

```
【効果的だったフレーズ】
・「ズボラな人ほどフォルダで仕分けることはやめたほうが良いです」
・「面倒くさいと感じる原因は、手間の多さです」
・「スワイプだけならいけるって人は、そこがあなたにとってちょうど良い管理方法」

【構成のポイント】
・最初に「使わないものを非表示」で画面をシンプルに
・中盤で「設定」を済ませる
・後半で「実際の運用」を見せる
・最後に「応用」を任意で紹介
```
