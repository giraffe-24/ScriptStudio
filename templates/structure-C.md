# 台本テンプレート C：まとめ紹介型

> 複数の項目を網羅的に紹介する型
> 例：Amazonプライム特典8選、Googleセキュリティ6選

---

## 適用条件

```
・複数の項目（5-10個）を紹介する
・各項目は独立している（順序性がない）
・「知らないと損」「〇〇選」のような切り口
・各項目の説明は短め（1-2分/項目）
```

---

## 構成

```
【導入】約350字（10-15%）
  │
  ├── 自己紹介（固定）
  ├── 挑発フック「〜、まさか〜だけで満足してませんか？」
  ├── 現状の問題提起「実は〜には〇個の〜がある」
  ├── 価値リスト（全項目を列挙）
  └── 宣言「今日は特に〜な〇つを厳選して紹介します」

【本題】約1800-2500字（70-75%）
  │
  ├── 項目1
  │     ├── 結論（1文で何ができるか）
  │     ├── 使い方/始め方（簡潔に）
  │     ├── メリット（具体的な数字があれば）
  │     └── 注意点（あれば）
  │
  ├── 項目2
  │     └── （同じ構造）
  │
  ├── 項目3〜N
  │     └── （同じ構造）
  │
  └── 番外編（任意）
        └── 「1分以上かかるけど超オススメ」など

【まとめ】約300字（10-15%）
  │
  ├── 全体の価値「〜は〇〇、△△、□□など、いろんなサービスがお得になります」
  ├── 視聴者への提案「この中から1つでも〜」
  ├── LINE誘導（任意）
  ├── 行動喚起
  └── 定型締め（固定）
```

---

## 導入テンプレート

```
「楽」を極めて最大の結果を出す、効率化オタクのあらきりです。

【サービス名】の【料金/条件】、払ってますよね?
でもその【料金】、まさか「【最も基本的な特典】」だけで満足してしまってませんか？

確かに、【基本特典の説明】は便利です。
でも実は、【サービス名】には全部で【数字】個の特典があるんです。

【全特典リスト】
・〇〇
・〇〇
・〇〇
（全て列挙）

今日は、その中でも特に「知らないとぜったいに損する」特典を【数字】つ厳選して、お話ししようと思います。
```

## 本題テンプレート（1項目）

### 基本パターン（短め・100-150字）

```
【見出し】項目名

【結論】
〇〇は、【何ができるか】特典/機能です。

【使い方】
【簡単な始め方】だけで使えます。
【便利な機能】があるので、【メリット】。

【対象者】
【こういう人】におすすめのサービスです。

【注意点（あれば）】
ただし【注意点】なので、【対処法/代替案】。
```

### 詳細パターン（長め・200-300字）

```
【見出し】項目名

【結論】
〇〇は、【何ができるか】サービスです。

【使い方/メリット】
【具体的な使い方】で、好みの【コンテンツ】が【どう使えるか】ので、【ユースケース】に最適です。

【具体的な数字】
【対象者】なら、年間で【金額/数量】が【効果】計算です。

【注意点】
注意点は、【制限/デメリット】ことです。
【その制限の説明】。

【代替案/アップグレード】
【別の選択肢】なら、【料金】の「【上位プラン】」へのアップグレードが必要です。

【まとめ】
【ユースケース】をする人にぴったりなサービスです。
```

### 番外編パターン

```
【見出し】番外編：〇〇

【前置き】
【条件】なんですけど、超オススメの〇〇もここで紹介します。

【結論】
〇〇において、一番【効果的/重要】な〇〇が「【項目名】」です。

【説明】
これは、【詳しい説明】のことです。

【始め方】
設定はとても簡単で、【手順の概要】から案内に沿って進めるだけです！

【補足】
詳しい設定方法は別の動画で紹介してるんで、参考にしてみてください。

【メリット】
一度設定してしまえば、【効果】ので、ズボラな人にこそ向いています。
```

## まとめテンプレート

```
【全体の価値】
〇〇の特典は、【カテゴリ1】、【カテゴリ2】、【カテゴリ3】など、いろんなサービスがお得になります。

【視聴者への提案】
【料金/条件】でこれだけの特典がついてくれば、今利用中の【競合カテゴリ】を見直すこともできるはずです。
この中から1つでも、自分の生活に合うものを見つけてみてください。

【LINE誘導（任意）】
公式LINEでは、これまで動画で紹介した便利なツールをまとめて配布しています。
LINEを登録すれば無料で受け取れますが、こちらのプレゼントは予告なく終了する可能性があります。
ぜひこの機会に早めに受け取ってください。

【定型締め】
このチャンネルでは「楽」を極めるために私が検証し、本当に効果があった仕組みだけを紹介しています。

大人の教養として『賢くサボる』を知りたい方は、ぜひチャンネル登録と高評価をお願いします。

日常に潜む「無駄な手間」や｢これも知りたい｣などのリクエストも、遠慮なくコメントで教えてください。

それでは、また次回の動画でお会いしましょう。さようなら。
```

## セクション見出しの例

```
【シンプルに項目名】
・Amazon Photos
・Amazon Mastercard
・Prime Video

【番号付き】
・1. アカウントのログイン履歴をチェックする
・2. パスワードが漏洩していないかを確認する
・3. 不要なアプリのアクセス権を解除する

【番外編】
・番外編：2段階認証をオンにする
・番外編：Googleパスワードマネージャもおすすめ
```

## この型の成功パターン

### Amazonプライム台本から抽出

```
【効果的だったフレーズ】
・「まさか『送料無料』だけで満足してしまってませんか？」
・「実は、Amazonプライムには全部で14個の特典があるんです」
・「月600円でこれだけの特典がついてくれば」

【構成のポイント】
・導入で「損してるかも」という気づきを与える
・全項目を最初にリストで見せる（何が得られるか約束）
・各項目は結論→使い方→注意点の短いサイクル
・番外編で「これも知っておくと良い」を追加
・まとめで「見直し」を提案
```

### Googleセキュリティ台本から抽出

```
【効果的だったフレーズ】
・「『自分は大丈夫』と思って何もしていない人は、騙す側にとってネギを背負った鴨」
・「でも安心してください。こうした被害の多くは、ちょっとした対策でリスクを大きく減らすことができます」
・「全部を一気にやろうとしなくても大丈夫です」

【構成のポイント】
・「1ステップ60秒」など時間の短さをアピール
・各項目の手順を具体的に（「〜をタップ」レベル）
・「番外編」で1分以上かかるものを別枠に
・まとめで「1つでもやれば」とハードルを下げる
```

## 項目数の目安

| 動画の長さ | 項目数 | 1項目あたり |
|------------|--------|-------------|
| 5分 | 3-4項目 | 約1分 |
| 8分 | 5-6項目 | 約1分 |
| 10分 | 6-8項目 | 約1分 |
| 12分以上 | 8-10項目 | 約1分 |

※導入・まとめで約2-3分使うため、本題は残り時間で配分
