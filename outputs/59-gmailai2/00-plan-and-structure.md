# 企画・構成：GmailのAI設定、確認してますか？2つだけチェック

> このファイルは Supabase 上の元データ消失にともない、`manifest.json` / `plan.json` から復元したものです。

---

## 企画

| 項目 | 内容 |
|------|------|
| タイトル | GmailのAI設定、確認してますか？2つだけチェック |
| ステータス | scripting |
| テーマ選定 | market（市場分析ベース） |
| 想定尺 | 8〜10分 |

### ターゲット視聴者

50代会社員。Gmailは仕事でもプライベートでも毎日使っている。最近ニュースでAIの話題をよく見かけ、自分のメールも読まれているのでは？と漠然と不安を感じている。設定を変えたいが、どこを触ればいいかわからず放置している。

### 視聴者の悩み（ペイン）

Gmailを普通に使っているだけなのに、知らないうちにAIに個人情報を読み取られているのではないかという漠然とした不安。確認したいが、設定画面のどこを見ればいいかわからない。

### 約束（プロミス）

動画を見終わる頃には、GmailのAI関連設定2か所を自分の目で確認でき、「ちゃんと把握している」という安心感を得られる。

### キーポイント

- Googleアカウントの「スマート機能」設定の確認方法
- Geminiアプリとの連携設定の確認方法
- 設定をオンのままにするか、オフにするかの判断基準

### 競合との差別化

競合動画は「怖い」「吸い取られている」と不安を煽る傾向がある。本動画は不安煽りを避け、「確認して自分で選ぶ」という安心感を軸にする。また、オンにするメリットも公平に伝え、オフ一択にしない点で差別化。

---

## 構成（structure-B：機能紹介型）

| パート | セクション | 内容 |
|--------|-----------|------|
| 導入 | 問いかけフック〜宣言 | 設定を確認せず使っている現状。不安の正体は「わからないこと」。今日確認する2つの設定を予告 |
| 本題1 | スマート機能の設定を開く | データとプライバシーの画面から「スマート機能」を確認。スマート機能とは何かを噛み砕いて説明 |
| 本題2 | スマート機能をオンにする人・オフにする人 | オン（予測変換・スケジュール提案）とオフ（解析を止める）のメリット。どちらが正解ではなく自分で選ぶ |
| 本題3 | Geminiアプリとの連携設定を開く | データとプライバシーから「Geminiアプリのアクティビティ」を確認。Geminiがメールを読む条件を説明 |
| 本題4 | Gemini連携をオンにする人・オフにする人 | オン（要約・返信案）とオフ（読ませない）のメリット。判断基準は便利さと安心感のバランス |
| まとめ | 確認したら、それで終わりでいい理由 | 毎日チェックは不要。一度確認すれば自分で選んだ状態。不安の正体は「知らないこと」だった |

> 注意：Googleアカウント／Gemini の設定画面の名称・場所は時期や端末で変わりやすい。台本では `config/quality.md`「製品・アプリの事実」に従い、断定せず「画面上で確認して」と逃がす。
