# 台本（ドラフト）｜Windowsでファイル名を一括整理｜PowerShellはコピペで十分

> 読み込みメモ（エージェント／人間向け）
> - 視聴者想定: 40〜60代中心。黒い画面やパスの扱いが初めての回でも追えるよう、手順は省略せず、キー操作は「Ctrl」と読むなど噛み砕く（本文では能力や習熟度を下げるラベル遣いはしない。`config/quality.md` 参照）。
> - 体裁: 口述用トーク原稿。操作は画面収録で見せる想定だが、音声だけでも追えるよう画面のどこを見るかを言葉で補う。
> - SSOT: `config/voice.md`・`config/quality.md`・`config/brand.md`（定型締め・冒頭自己紹介は brand どおり）
> - 方針: コマンドと AI 依頼文は完成形のコピペ。手入力は避ける。料金・無料枠は各自の画面と公式表示に任せる。

## 【導入】右クリック地獄から抜けたい人へ

「楽」を極めて最大の結果を出す、効率化オタクのあらきりです。

ダウンロードの中身が増えて、気づいたら名前がバラバラで、一枚ずつ右クリックから名前の変更を繰り返すのがだるい、という感覚、わかります。私もズボラなので、人間が手で直し続けるのは向いていません。

今回の動画でやることは一つに絞ります。Windows に最初から入っている PowerShell という黒い画面に、あらかじめ用意した文章を貼って、フォルダの中のファイル名を、一度にそろえる、という流れです。プログラミングのお勉強ではありません。覚えるのではなく、コピーと貼り付けだけで再現できる形にします。

ただし、ここから先は手順の塊が細かく感じるところです。具体的には「フォルダの住所のコピー」と「黒い画面への貼り付け」を、一緒にゆっくりやります。このパートを飛ばすと、あとで「貼ってない」「場所が違う」となりやすいので、収録でも間を取って見せます。初めての方は、本物の大事な写真や仕事の資料が入ったフォルダではなく、デスクトップに「練習用」フォルダを一つ作って、その中に消えても困らないコピーのファイルを数個入れてから試してください。準備ができたら、一緒に進みましょう。

## 【本題】先に安全だけ約束する｜練習用フォルダのつくり方

本題に入る前に、安全のルールだけ三つです。

一つ目、今日初めての方は練習用で試す。デスクトップの何もないところで右クリックして、新規作成からフォルダーを選び、名前を「リネーム練習」など分かりやすいものにします。その中に、メモ帳で適当な文書を数個作ってもいいし、いらない写真のコピーを入れてもいいです。大事な原本は触らない、と決めてください。

二つ目、それでも不安なら、練習フォルダごと複製しておく。練習フォルダを一回クリックして選んだ状態で、Ctrl キーを押しながら C を押してコピーし、デスクトップの空いているところで Ctrl キーを押しながら V を押して貼り付け、「リネーム練習 のコピー」のようなフォルダがもう一つできれば成功です。万が一のときはこのコピー側だけを触ります。

三つ目、黒い画面に貼る長い文は、必ずこの動画の概要欄か、画面に出しているブロックからコピーします。自分で一文字ずつ打たない、これが今日の約束です。

## 【本題】PowerShell とは何か｜黒い画面の正体

結論だけ先に言います。PowerShell は、マウスではなく文字の命令で作業を進めるための、Windows 公式の道具です。見た目は黒い窓で、映画のハッカーみたいに見えるかもしれませんが、中身は「コピーした命令を一行ずつ実行する場所」です。

あわせて言葉を一つ。最近のパソコンにある「Windows ターミナル」は、いわば下敷きのノートのような枠です。その枠の中に、PowerShell という種類のページを開くこともあれば、昔からあるコマンドプロンプトという種類を開くこともあります。今日使う命令は PowerShell 向けです。ページの種類が違うと、貼った命令が動かないことがあるので、最初に「PowerShell と書いてある場所でやる」と頭に入れてください。

## 【本題】PowerShell を開く｜最初の一歩

おすすめは、いちばん迷子にならないやり方から説明します。

画面の左下にある、四つの青い四角が重なったマーク、Windows マークを、マウスで一度クリックします。メニューが開いたら、キーボードでゆっくり PowerShell と打ちます。英字で構いません。大文字小文字は気にしなくて大丈夫です。

一覧に「Windows PowerShell」や「PowerShell」という名前のアプリが出てきたら、それをマウスでクリックします。黒い窓が開けば、とりあえず第一段階クリアです。管理者として実行、のような文字は、今日は選ばなくて構いません。普通に開いてください。

もう一つの行き方を知っておくと便利です。エクスプローラーで、いま作業したいフォルダを開いた状態で、フォルダの中の何もないところ、またはフォルダアイコン自体を、Shift キーを押したまま右クリックします。メニューに「ターミナルで開く」やそれに近い文字がある場合は、それを選びます。枠が開いたら、タブのところに PowerShell と書いてあるか確認してください。コマンドプロンプトとだけ書いてあるときは、この動画の命令がそのまま動かないことがあるので、上のメニューや下向きの矢印から新しいタブを PowerShell に切り替えてください。Windows の版によって言葉が少し違っても、「PowerShell という単語にたどり着く」ことを目標にしてください。

## 【本題】フォルダの「住所」をコピーする｜いちばん大事なパート

ここからは、パスと呼ばれる住所のコピーを扱います。黒い画面に慣れていない日でもつまずきにくいよう、操作だけに集中して説明します。「パス」とは、そのフォルダがパソコンの中でどこにいるかを表す住所のようなものです。手で打ち直すと誰でもミスしやすいので、今回はWindowsの機能にコピーさせます。方法は二通り覚えてください。

方法の A。エクスプローラーで、作業したいフォルダの中身を表示した状態にします。画面上部に、ホームやギャラリー、次にパンくずのような角ばった名前が並ぶ欄があります。その左側や、行全体をクリックすると、長い一行の住所が青く全部選ばれることがあります。選ばれたら、キーボードの Ctrl キーを押しながら C を押してコピーです。もしクリックしても角ばった名前のまま変わらないときは、Ctrl キーを押しながら L を押してみてください。住所の入力行にカーソルが移り、文字が一行選ばれるタイプの画面もあります。そうなったら同じく、Ctrl を押しながら C でコピーです。

方法の B。コピーしたいフォルダのアイコンそのものを、Shift キーを押しながら右クリックします。メニューの中に「パスとしてコピー」という項目が出ることがあります。それを選ぶと、先頭のと末尾に少し特殊な記号が付いた状態でコピーされることがあります。今日お渡しする命令では、すでに cd の横に引用符を用意しているので、パスとしてコピーで両端に余分な記号が付いてしまった場合は、貼り付けたあとに記号が重なっていないかだけ、ゆっくり目で確認してください。

コピーできたか不安なときは、デスクトップでメモ帳を開き、中を一度クリックしてから Ctrl を押しながら V を押してみてください。住所らしき英数字とバックスラッシュの羅列が並べば、コピーは成功です。試して問題ありません。

## 【本題】PowerShell に貼り付ける｜cd の一行

黒い画面の、点が光る行、いわゆるカーソルのあるところを、マウスで一度クリックします。貼り付けのやり方は環境で二パターンあります。右クリックしただけで貼り付く古い型の PowerShell もあれば、Ctrl キーを押しながら V で貼る型もあります。どちらかで文字が現れたら成功です。何も起きなければ、もう一方も試してください。

いよいよ場所の移動です。画面と概要欄に、次の形の一行があります。引用符は半角のままです。メモ帳から拾ったときに、かぎ括弧が別の形に変わっていないかだけ気をつけてください。

    cd "ここにフォルダのパスを貼り付け"

やり方はこうです。引用符の直後から、閉じる引用符の手前までを、マウスでなぞって青く選択します。そこに、さきほどコピーした住所を、Ctrl を押しながら V で上書き貼り付けします。左の cd とスペースと最初の引用符、右の閉じる引用符はそのまま残してください。住所の中に、すでに引用符が入ってしまっていたら、記号が二重になっていないか見直します。

内容を確認したら Enter キーを一回押します。エラーが出ずに次の行に進めば、フォルダの移動は成功しやすいです。もし真っ赤な文字で注意が出たら、そのまま次に進まず、一回深呼吸して、引用符の対応を確認するか、概要欄の AI 用テンプレに進んでも大丈夫です。自分一人で直そうとしなくて結構です。

## 【本題】一覧を出す｜いま正しい場所にいるか確認

場所が合っているか、次の一行を貼って確認します。これも打たずにコピーです。

    Get-ChildItem

Enter を押して、ファイル名らしき名前がずらっと並べば、そのフォルダの中身を見ている状態です。何も出ないときは、練習フォルダの中にファイルが本当にあるか、エクスプローラーで一度見てから戻ってきてください。

## 【本題】リネームの予行と本番｜-WhatIf を必ず先に

いよいよ名前の一括変更です。説明欄には、日付の接頭辞の例を載せています。次の接頭辞の日付部分は、収録の日付に合わせて差し替えてください。画面に出ている完成形を優先します。

まず予行です。-WhatIf が付いている命令は、練習であって本番ではありません。実際の名前はまだ変わりません。次の一行を貼って Enter です。

    Get-ChildItem -File | Rename-Item -NewName { '2026-05-12_' + $_.Name } -WhatIf

画面に、もし変えるならこうなる、という説明が英語や日本語まじりで並びます。違和感がなければ、同じ内容から -WhatIf だけを消した次の一行を貼ります。同じフォルダで二回実行すると、接頭辞が二重に付くので、本番は一回だけにしてください。

    Get-ChildItem -File | Rename-Item -NewName { '2026-05-12_' + $_.Name }

別のよくある形として、ファイル名のスペースをアンダースコアに置き換えたい場合は、予行と本番の二行を、この順で貼ります。

    Get-ChildItem -File | Rename-Item -NewName { $_.Name -replace ' ','_' } -WhatIf
    Get-ChildItem -File | Rename-Item -NewName { $_.Name -replace ' ','_' }

いま開いているフォルダの、すぐ下の段のファイルだけが対象です。下のフォルダの中まで一気にたどる設定は、最初は付けません。知らないうちに別の場所まで変わってしまう事故を避けるためです。

## 【本題】うまくいかなかったとき｜人間でも AI でもいい逃げ道

まず人間側のチェックです。練習フォルダのコピーを残していたら、いちばんラクなのは、やり直し用のコピーをもう一度作ることです。黒い画面での取り消しに頼りすぎず、手元に無改造のコピーを置いておくのが、ズボラにとって一番の保険です。

次に AI です。ブラウザを開き、アドレスバーに claude.ai と打って Enter、または gemini.google.com と打って Enter します。初めてのサービスなら、画面の案内に従ってアカウントを作るかログインします。無料で試せる範囲は、そのときの画面の表示と公式の案内を見て判断してください。

ログインできたら、チャットの入力欄を一度クリックします。貼る文章は、この動画の概要欄と同じものを使います。ナレーションでは読み上げませんので、画面に全文を出すか、概要欄を開いたままにしてください。まず次のかたまりを、そのままコピーします。

    あなたはWindowsのPowerShellに詳しいアシスタントです。次の条件だけ守って、実行すべきコマンドを「コピペできる1つのコードブロック」だけで出力してください。解説は短くてよいです。

    条件:
    - 対象は「指定フォルダの直下のファイルだけ」。サブフォルダは再帰しない。
    - 必ず最初に -WhatIf 付きの予行コマンド、その次に -WhatIf なしの本番コマンドを、別ブロックで出す。
    - 危険な一般化（全ドライブなど）は書かない。

    やりたいこと:
    [ここに一文で。例: ファイル名の先頭に 2026-05-12_ を付ける／スペースをアンダースコアにする]

    フォルダのフルパス:
    [ここにエクスプローラーでコピーしたパスを貼る]

貼ったあとで、角括弧から角括弧までを含めて消して、自分の言葉と、さきほどコピーしたフォルダの住所だけを入れ直します。角括弧の記号そのものは残さないでください。キーボードで長い文章を考えなくてよいです。やりたいことの行は、画面に出す短文の例をそのまま使っても構いません。

送信ボタンを押したら、返ってきた答えの中から、コードの塊だけをコピーします。余計な説明文まで黒い画面に貼らないよう、まずメモ帳に貼って、命令だけを選んでから PowerShell に持っていくと安全です。AI に頼むときも、クレジットカード番号や他人の個人名、会社の機密がファイル名に入っているフォルダは貼らないでください。ダミーの練習フォルダで十分です。返ってきた命令も、必ず -WhatIf のブロックから先に実行してください。

## 【まとめ】コピペで持ち歩く最低限

今日の本質は三つだけです。PowerShell と書いてある黒い画面を開く。フォルダの住所はアドレスバーか、パスとしてコピーで持ってきて、手では打たない。名前を変えるときは -WhatIf の予行を踏んでから、本番は一回だけ。

整理の哲学までは背負わなくて大丈夫です。フォルダは深掘りしすぎない、ファイル名の型を一つ決める、くらいで続きます。概要欄のブロックをメモ帳に貼っておき、日付の接頭辞だけ時々書き換える、そんな使い方で十分です。この黒い画面は毎日使う道具ではなく、「名前がぐちゃっとした日の救急道具」だと思ってください。
