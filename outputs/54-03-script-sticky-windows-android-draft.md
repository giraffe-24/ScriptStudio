## 【導入】

「楽」を極めて最大の結果を出す、効率化オタクのあらきりです。

Windows のパソコンでちょこちょこメモするんだけれど、外出先にはスマホだけ、という方いますよね。私もそうなんですが、スマホ側は Android を使っていて、PC の付箋に書いたことをそのまま一覧で見たいだけなのに、どこから手をつければいいのか、ちょっとだけ迷うことがありました。

今回の動画では、付箋アプリの細かい便利機能を全部紹介するわけではなく、「同期して、同じ一覧が Android でも見える状態になるところまで」に絞ってお話しします。

あらかじめお伝えしますが、今回お見せするのは Windows と Android の組み合わせです。Mac や iPhone の手順は別の世界になりますので、この動画ではそこには触れませんね。

ってことで、準備ができたらパソコンと Android を手元に置いた状態で、一緒に確認していきましょう。

---

## 【本題】今日やらないこと／今日のゴール

まず「やらないこと」をはっきりさせます。

今日は、メモ術の講座でもなければ、付箋の色の使い分けを哲学する回でもありません。ほかのメモアプリとどれが正しいかみたいな話もしません。

ゴールは一つだけです。Windows で付箋として書いた内容が、Android でも同じ Microsoft アカウント経由で一覧として確認できる状態になっていること、が今日のゴールです。はっきり言うと、Android 側では「Sticky Notes と書かれただけの単体アプリを入れればそれだけで済む」と公式が書いているわけではなく、一覧に触れる主な経路は Microsoft OneNote の中の「付箋」、英語表記では Sticky Notes と出る入口だ、という説明になります。メニューの位置や名前は、アプリの版や端末で変わっても不思議じゃありません。「自分の画面上で一覧に触れられるか」を、こちらでも確認するところまでを狙います。

---

## 【本題】Windows 側｜ここだけ押さえる3点

では PC 側です。付箋って、正式には Sticky Notes と呼ばれるアプリですね。まずはそれを開きます。スタートメニューからでも、検索からでも、普段のやり方で一つに決めてしまうと、説明も撮影もラクです。

開けたら確認してほしいのは、大きく三つです。

一つ目、付箋がちゃんと起動していること。当たり前に聞こえるかもしれませんが、ここから先は「アプリが開いている」前提で話を進めます。

二つ目、Microsoft アカウントでサインインできているか。アカウントの確認方法は、Windows のバージョンや付箋の画面で少し差が出ます。なので、画面上に表示されている「今、どのアカウントでログインしているか」という表示を、自分の目で一度見てください。ここは画面上で確かめられる範囲でしか言い切れないので、その言い方だけは覚えておいてくださいね。

三つ目、テスト用に、短い文を一つ、新しい付箋として書いておきましょう。「テスト」だけでもいいので、あとで Android 側に同じ文字列が出てきたか確認するための目印です。

ここまで終わったら、いったん PC はそのままで大丈夫です。

---

## 【本題】Android 側｜OneNote から一覧へ（見え方は実機で確認）

続いて Android です。結論から言います。Google Play を開いて「付箋」だけの名前の単体アプリを入れれば完結する、とは今日は想定していません。

Microsoft が案内している中心の流れは、開発元が Microsoft と分かる公式の OneNote を入れ、そのアプリの中にある Sticky Notes、日本語だと「付箋」と表示されることがある入口から、PC と同期した一覧に触れる、という形です。

まず Play ストアで OneNote をインストールし、起動してさっき PC の付箋で使ったのと同じ Microsoft アカウントでサインインしてください。これがずれると一覧が並んでも自分の話にならなくなりますので、細かくてもチェックしましょう。

Microsoft の公式ヘルプも、画面上は「OneNote を開いて、右下角あたりで Sticky Notes をタップ」といった案内になります。ただし私が話したあとにアプリやメニューの並びが変われば、文言やアイコンの場所も別ものになるので、この動画を見ている方は自分の画面上で「自分の環境での付箋の入り口」を探していただいて、収録側も収録その日に一度実機を開いて確認してから収録しましょう。見当たらないときは、ストアで OneNote をアップデートしてからまた開いてみる、というのが王道です。

同期がすぐに反映されないこともあります。数十秒から少し長めに待ってから、更新できる操作があれば試すくらいの気持ちでいてください。必ず即時、とは私からは言い切れません。

余裕があれば聞いておくと、Microsoft Launcher をホームに使っている方は、公式の説明では左端のフィードなどで付箋に触れる設定もある、といった案内もあります。Launcher を普段から替えていない方は、まず OneNote の経路だけで十分です。

---

## 【本題】同期の確認〜試し書き

さて、OneNote 側の付箋の一覧を開ける画面まで来たら、さっき PC で書いたテスト用の短文が、もう見えていますか。

もし見えていたら、その時点でもう「今日やりたかったことの核」はクリアです。拍手。

もしまだだったら、OneNote をいったん止めてから開き直す、Wi-Fi とモバイルデータの状態を確認する、ストアで OneNote をアップデートする、ログイン情報を見直す、という順で丁寧にいきましょう。

余裕がある方は、Android 側で一行だけ追記してみて、そのあとパソコン側の付箋一覧を確認する、という片方向だけのテストでも構いません。尺や撮影の都合があれば、ここは飛ばしても大丈夫です。

---

## 【本題】うまくいかないとき｜チェックは三つだけに絞る

トラブルシュートを長くすると動画全体がぐだっとしますので、チェックリストは 三項目だけ にします。

一つ目、やはりこれです。PC の付箋と Android の OneNote が、同じ Microsoft アカウントになっているか。両方の設定やアカウント情報を並べられるなら並べて、心当たりがないズレがないか見る。

二つ目、OneNote 本体と Windows 側の付箋を、それぞれ自分のタイミングでアップデートしておく。公式のトラブルシュートでも、まず最新版に、という話が何度も出てきます。

三つ目、サインインの切れ。どちらかでログアウトしたり、アカウント変更したりしていたら、再接続が必要です。再サインインの手順は、画面ひとつの範囲で終わるように自分でルートを決めてください。

ほかにも原因はあり得ますが、この三つがクリアになると不思議なくらいハマることが多いです。それでも無理なら、Microsoft のサポート記事やコミュニティの情報をあわせて見るくらいの姿勢でOKです。特定のサービスの悪口に逃げないようにだけ注意してほしいです。私たちはあくまで自分の環境で再現することに集中しましょう。

## 【まとめ】

というわけで、今回やったことを短くおさらいします。

Windows の付箋は、気軽なメモ向きのアプリでありながら、Android では Microsoft OneNote の中から同じアカウントの付箋一覧に触れられるので、コンビニの前で急に思い出したメモを見返す、みたいな場面で重宝します。

ただし、付箋はあくまで短期のメモ向きです。心の中で「ここに人生の全部を預ける」と決めつけないでください。大事な原本は、みなさんが普段から信頼している保管の仕方と併用するのが安全です。

今日の時点で、Android の OneNote から付箋の一覧に入って、PC のテストメモが見えていれば、目的は達成です。ぜひ一度、ご自身の仕事や生活のワンシーンで試してみてください。
