# 台本（参考ドラフト）｜Windowsの付箋をAndroidで見る〜同期の確認まで

> - ソース: `02-structure-sticky-windows-android-8min.md` に沿った口述用の完成形です。「ドキュメント記事」ではなくトーク原稿です。

> - 尺の目安: 約8分収録想定。収録前にアプリ名・メニュー表記は実機の画面に合わせて差し替えてください。

## 【導入】

「楽」を極めて最大の結果を出す、効率化オタクのあらきりです。

Windows のパソコンでちょこちょこメモするんだけれど、外出先にはスマホだけ、という方いますよね。私もそうなんですが、スマホ側は Android を使っていて、PC の付箋に書いたことをそのまま一覧で見たいだけなのに、どこから手をつければいいのか、ちょっとだけ迷うことがありました。

今回の動画では、付箋アプリの細かい便利機能を全部紹介するわけではなく、「同期して、同じ一覧が Android でも見える状態になるところまで」に絞ってお話しします。

あらかじめお伝えしますが、今回お見せするのは Windows と Android の組み合わせです。Mac や iPhone の手順は別の世界になりますので、この動画ではそこには触れませんね。

ってことで、準備ができたらパソコンと Android を手元に置いた状態で、一緒に確認していきましょう。

---

## 【本題】今日やらないこと／今日のゴール

まず「やらないこと」をはっきりさせます。

今日は、メモ術の講座でもなければ、付箋の色の使い分けを哲学する回でもありません。ほかのメモアプリとどれが正しいかみたいな話もしません。

ゴールは一つだけです。パソコンの付箋に書いた内容が、Android 上で、Microsoft の公式のアプリの経路を通じて一覧として見える状態になっているかを、こちらでチェックできるようにすること。ここまでできれば、今日の勝ち、と考えてください。

---

## 【本題】Windows 側｜ここだけ押さえる3点

では PC 側です。付箋って、正式には Sticky Notes と呼ばれるアプリですね。まずはそれを開きます。スタートメニューからでも、検索からでも、普段のやり方で一つに決めてしまうと、説明も撮影もラクです。

開けたら確認してほしいのは、大きく三つです。

一つ目、付箋がちゃんと起動していること。当たり前に聞こえるかもしれませんが、ここから先は「アプリが開いている」前提で話を進めます。

二つ目、Microsoft アカウントでサインインできているか。アカウントの確認方法は、Windows のバージョンや付箋の画面で少し差が出ます。なので、画面上に表示されている「今、どのアカウントでログインしているか」という表示を、自分の目で一度見てください。ここは画面上で確かめられる範囲でしか言い切れないので、その言い方だけは覚えておいてくださいね。

三つ目、テスト用に、短い文を一つ、新しい付箋として書いておきましょう。「テスト」だけでもいいので、あとで Android 側に同じ文字列が出てきたか確認するための目印です。

ここまで終わったら、いったん PC はそのままで大丈夫です。

---

## 【本題】Android 側｜インストールから一覧まで

続いて Android です。Google Play を開いて、Microsoft が提供している 付箋、英語表記の場合は Sticky Notes という名前のアプリを探して、インストールしてください。ストアに載っている表記は時期で少し変わることがあるので、「Microsoft さんの公式の付箋アプリかどうか」を見極めてから入れるのが安心です。

インストールが終わったらアプリを起動して、さっき PC で確認したのと同じ Microsoft アカウントでサインインします。ここ、本当に大事なので、もう一度言います。PC と Android で、アカウントがズレていると、今日やりたいことは成立しません。 メールアドレスが同じ系列か、設定画面で最後の数桁が合うか、映せる範囲で軽く照合しておくとよいです。

サインインが済むと、だいたいはメモの一覧のような画面に入ります。権限のダイアログが出たら、付箋を普通に使うのに必要そうなものだけ、テキストの指示にしたがって許可してください。ここで「全部ブロック」を選んでしまうと、後で説明できないタイプのつまずきになることがありますので、ひと工夫という感じで大丈夫です。

それから大事なことを一つだけ。同期って、環境によってはすぐ終わらないときがあります。今タップしたのに一覧がパッと変わらない、というときは、数十秒単位から、少し長めに待ってから、画面上で更新できる操作があればそれを試す、くらいの感覚でいてください。何十秒で必ず終わる、とは言い切れないので、そこだけ期待値をリアルにしておきましょう。

---

## 【本題】同期の確認〜試し書き

さて、Android の一覧を見てください。さっき PC で書いたテスト用の短文が、もう並んでいますか？

もし並んでいたら、その時点でもう「今日やりたかったことの核」はクリアです。拍手。

もしまだだったら、アプリをいったん止めてから開き直す、Wi-Fi とモバイルデータの状態を確認する、ログイン情報を見直す、という順で丁寧にいきましょう。

余裕がある方は、Android 側で一行だけ追記してみて、そのあとパソコン側の一覧を確認する、という片方向だけのテストでも構いません。尺や撮影の都合があれば、ここは飛ばしても大丈夫です。

---

## 【本題】うまくいかないとき｜チェックは三つだけに絞る

トラブルシュートを長くすると動画全体がぐだっとしますので、チェックリストは 三項目だけ にします。

一つ目、やはりこれです。PC と Android が、同じ Microsoft アカウントになっているか。 両方の設定やアカウント情報を並べられるなら並べて、心当たりがないズレがないか見る。

二つ目、アプリ本体と Windows 側のアップデートを一度、自分のタイミングで確認する。どのバージョンから使えない、まではここでは断定しませんが、「古いまま放置」は避けたいですよね。

三つ目、サインインの切れ。どちらかでログアウトしたり、アカウント変更したりしていたら、再接続が必要です。再サインインの手順は、画面ひとつの範囲で終わるように自分でルートを決めてください。

ほかにも原因はあり得ますが、この三つがクリアになると不思議なくらいハマることが多いです。それでも無理なら、コミュニティの情報やヘルプをあわせて見るくらいの姿勢でOKです。特定のサービスの悪口に逃げないようにだけ注意してほしいです。私たちはあくまで自分の環境で再現することに集中しましょう。

---

## 【まとめ】

というわけで、今回やったことを短くおさらいします。

Windows の付箋は、気軽なメモ向きのアプリでありながら、同じアカウントさえそろえば Android 側の公式アプリから一覧できるので、コンビニの前で急に思い出したメモを見返す、みたいな場面で重宝します。

ただし、付箋はあくまで短期のメモ向きです。心の中で「ここに人生の全部を預ける」と決めつけないでください。大事な原本は、みなさんが普段から信頼している保管の仕方と併用するのが安全です。

今日の時点で、Android の一覧に PC のテストメモが見えていれば、目的は達成です。ぜひ一度、ご自身の仕事や生活のワンシーンで試してみてください。
