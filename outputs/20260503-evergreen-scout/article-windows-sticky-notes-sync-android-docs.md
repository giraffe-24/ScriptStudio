## はじめに（この記事でできること）

- できること：PC の付箋で書いたメモが、Android 端末でも同じ Microsoft アカウント経由で一覧として見える状態になるための確認手順。
- やらないこと：メモ術全般・色設計講義・他アプリとの優劣議論・アプリごとの詳細レビュー。

> 留意：付箋は「短いメモ向き」のツールです。重要な資料のバックアップ専用にはせず、必要に応じて別の保管方法も併用してください。

## 事前にそろえるもの

- Windows PC（付箋アプリ／Sticky Notes が利用できる環境）
- Android 端末
- PC と Android の両方で同じ Microsoft アカウントでサインインできること（メールアドレスが一致しているか、後述の手順で確認）

---

## Windows 側で確認すること

| 手順 | 内容 |
|------|------|
| 1 | 付箋（Sticky Notes） を起動する（スタートメニューや検索から開くなど、利用環境で一つのパターンに固定すると説明しやすい）。 |
| 2 | Microsoft アカウントでサインインしているかを確認する（付箋内や Windows の設定画面に表示されるアカウント情報を参照。断定は自分の画面上で確認できる範囲に留める）。 |
| 3 | テスト用に 短い文を1件、新しい付箋として書いておく（後で Android に同じ内容が現れるか確認するため）。 |

---

## Android 側のセットアップ

| 手順 | 内容 |
|------|------|
| 1 | Google Play から、Microsoft 提供の 付箋（Sticky Notes） アプリをインストールする（ストア上の表示名に合わせる）。 |
| 2 | アプリを開き、PC と同じ Microsoft アカウントでサインインする。 |
| 3 | メモ一覧／リストが表示される画面まで進む。権限の確認ダイアログが出た場合は、必要なものだけ許可する。 |
| 4 | 端末や回線状況によっては、同期までに少し時間がかかることがある。すぐに表示されない場合は、しばらく待ってから一覧を更新する。 |

---

## 同期が取れているかの確認

- Android の一覧に、PC で書いた テスト用の短文 が表示されるかを見る。
- 必要に応じて、Android で1行追記したあと PC 側を開き直すなど、一方向だけでもよいので変化があるか確認する（尺や撮影次第で省略可）。

---

## うまくいかないときのチェック（最大3項目）

1. PC と Android で、同じ Microsoft アカウントになっているか（それぞれの設定またはアカウント表示で確認。映せる範囲でメールの末尾などを照合）。
2. アプリおよび Windows を最新に近い状態にしておく（Google Play と Windows の更新確認を自分の環境で行う。バージョンの断定はしない）。
3. どちらかでサインインが切れている可能性を疑い、一度サインアウト・再サインインを試す（手順は画面1つぶんに留める）。

> 本稿では、特定サービスとの比較や「だから危ない」といった煽りには立ち入りません。
